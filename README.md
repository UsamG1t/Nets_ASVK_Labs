# Лабораторные работы на тему «Сетевые протоколы в Linux»

Лабораторные работы направлены на закрепление практических навыков настройки
и администрирования сети на машинах с операционной системой Linux. 
Лабораторные работы могут быть интегрированы в лекционные курсы по изучению 
стека сетевых протоколов.

Данные лабораторные работы используются в рамках программы дополнительного
образования в курсах «Протоколы и методы управления и передачи данных в сети Интернет» 
и «Современные сетевые технологии», а также в курсах бакалавриата («Введение в сети ЭВМ»)
и магистратуры («Сети передачи и обработки данных») ВМК МГУ.

Для прохождения лабораторных работ используется разработанный образ 
виртуальной машины для VirtualBox на основе операционной системы ALT.
